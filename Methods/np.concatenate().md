## 2. np.concatenate([array1, array2])
Purpose: Glues multiple arrays together into one.

* Direction: By default, it stacks 1D arrays horizontally (end-to-end) to create one long ribbon of data.
* Syntax: Note the square brackets [] inside the parentheses; you must pass the arrays as a list.

Example:

# Creates one long array of 145 values (90 + 55)all_bills = np.concatenate([dinner, lunch])

| Function | Goal | Real-World Analogy |
|---|---|---|
| .concatenate() | Join Data | Taping two pieces of string together. |
